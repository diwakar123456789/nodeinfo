HTTP server library

In traditional server setup, one needs to use additional server software such as apache, spinner, CL – HTTP, GoServe, etc. to use OS connection properties such as HTTP, TCP/IP. But, node.js has built in function to access connection properties of an OS. And, for creating an HTTP connection, we can use built in HTTP library of node.js.

Supports NoSQL

NoSQL is the database system where data are managed with key values, graphs, documents, etc. It is scalable than traditional tabular database systems. Therefore, it is efficient to make real-time web application based on node.js.

Single Threaded Connection

Node.js establishes connection thread for the execution of each process and uses the same connection thread for the further processes if possible. After the execution of a process ends, the connection is not lost yet. So Node.js uses the same connection thread to execute any further processes. So the connection is faster and effective.

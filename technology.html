Java Script V8 Engine

V8 engine is an open source project by Google which executes JavaScript in Google Chrome browser. It was first released publicly on September 2, 2008 along with the chrome browser. In the node.js v8 engine is used in the server side.

Asynchronous or Non-blocking I/O

Asynchronous execution refers to execution of code that doesn't run in the sequence. So node.js allows the execution of the multiple processes at the same time and makes the overall operation faster.

npm package manager

npm is the package manager for node.js similar to composer of PHP, twisted of Python, event machine of Ruby. It helps to install and update packages for node.js application.

Introduction

Node.js is the framework to develop the real-time web based applications using JavaScript. It is used in developing both server side as well as the client side web applications. It places the continuous connection with the server and hence the real-time applications made with the node.js are fast, effective and dynamic. Its unique features make it most discussed topic among the programmers and other enthusiast.

Advanced

Node.js® is a platform built on Chrome's JavaScript runtime for easily building fast, scalable network applications. Node.js uses an event-driven, non-blocking I/O model that makes it lightweight and efficient, perfect for data-intensive real-time applications that run across distributed devices. It is open source and is available in all platforms i.e. windows, Linux and mac. The current stable version of node.js is 0.12.0.
